
SQLGen
===========

	SQL Query generator from website (such as w3schools.com) to help learn sql

usage
-------------

```python
sqlgen.py MY_TABLE
```

-----------

![Example](https://cumulus13.github.io/img/sqlgen.gif "example use w3schools.com")[zoom](https://cumulus13.github.io/img/sqlgen_large.gif)

- make_colors
- pydebugger
- clipboard
- unidecode
